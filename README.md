# Мобильное приложение для клинических исследований

### Описание

Мобильное приложение предназначено для клинических исследований медицинских изображений в тиреоидологии. Основные возможности приложения:

- **Анализ загруженных снимков УЗИ**
- **Выделение проблемных участков на изображениях**
- **Отображение рекомендаций на основе анализа**

Приложение разработано для использования пациентами, нуждающимися в диагностике заболеваний щитовидной железы.
